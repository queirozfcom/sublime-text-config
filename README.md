# sublime-text-3-config

# To install

## Sublime-text 3

```
$ git clone git@github.com:queirozfcom/sublime-text-config.git /tmp/sublime-stuff && mv /tmp/sublime-stuff  ~/.config/sublime-text-3/Packages/User
```
